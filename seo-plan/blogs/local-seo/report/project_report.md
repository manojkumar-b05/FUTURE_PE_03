# Project Objective

The objective of this project was to design an AI-powered SEO Content System using Prompt Engineering techniques. The project demonstrates how AI can assist businesses in creating search engine optimized content, improving local SEO, and building a structured content marketing strategy.

The selected business for this project is **Maverick Cafe**, located in Padur, Chennai.
